Welcome to the Blacksun books selection! We have compiled this list so the average Internet user can become enlightened in all aspects of the computer security field. Before you begin surfing the links we have to give a big thanks to Kript0n and Void Networks for compiling the links section. If it wasn't for the Void team this compilation wouldn't have been possible, well possible but not successful. If you have found a dead link or have a url not listed here, please email Mikkkeee.

Note: Blacksun does not host any ebooks, we only offer links so dont' bug us about illegal ebooks. Also, some of the books are in PDF format so you will need Adobe Acrobat, other than that enjoy your reading!
